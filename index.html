How to use

  • buttons

  • Let's go to see! 버튼 : 포트폴리오의 내용을 볼 수 있습니다.
  • Portfolio < /ji_yoon >버튼 : 포트폴리오 내용 안에 들어오시면 튜브 안에 Portfolio < /ji_yoon > 누르시면 다시 처음 화면으로 돌아가실 수 있습니다.
  • View More!! 버튼 : 포트폴리오 내용 안에 들어오시면 튜브에 View More!!을 누르시면 각각의 내용물을 볼 수 있습니다.

  • tube slide

  • 튜브에 마우스를 올리면 튜브가 돌아가는것을 멈출 수 있습니다.
  • 마우스가 튜브 밖으로 벗어나면 다시 튜브가 돌아갑니다.
  • * 혹시나 파도가 내용물을 다 덮을 경우 다시 새로고침 해주시면 내용물을 다시 보실 수 있습니다.
